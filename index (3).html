Nincs megadva

⚠ GitHub Pages-ről a Hostaway API csak proxy-n keresztül működik. Cloudflare Worker URL megadása szükséges.

Ha nincs proxy: a Kapcsolat tesztelés és a Lakások betöltése nem fog működni.
Proxy nélkül is lehet manuálisan árazni és Claude API-t használni.
